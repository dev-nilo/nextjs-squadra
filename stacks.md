# Stacks do Projeto

Este projeto utiliza as seguintes tecnologias, bibliotecas e ferramentas:

## Core Framework & Linguagem
- **Next.js**: 16.0.10 (App Router)
- **React**: 19.2.0
- **TypeScript**: 5.x
- **Node.js**: (Gerenciado via .nvmrc)

## Estilização & UI
- **Tailwind CSS**: 4.1.9
- **PostCSS**: 8.5
- **shadcn/ui**: Componentes de UI reutilizáveis (baseados em Radix UI)
- **Radix UI**: Primitivos de UI acessíveis (Accordion, Dialog, Popover, etc.)
- **Lucide React**: Biblioteca de ícones
- **cva (class-variance-authority)**: Variantes de componentes
- **clsx / tailwind-merge**: Utilitários para classes CSS condicionais
- **next-themes**: Gerenciamento de temas (Dark/Light mode)
- **tailwindcss-animate**: Utilitários de animação para Tailwind
- **Recharts**: Biblioteca de gráficos

## Formulários & Validação
- **React Hook Form**: Gerenciamento de estado de formulários
- **Zod**: Validação de esquema (schema validation)
- **@hookform/resolvers**: Integração entre Zod e React Hook Form

## Backend & Dados
- **Supabase**: Backend-as-a-Service (Auth, Database, Realtime) - `@supabase/supabase-js`, `@supabase/ssr`
- **Firebase**: (Presente nas dependências, versão 12.8.0)
- **date-fns**: Manipulação de datas

## Outras Bibliotecas
- **Sonner**: Componente de Toast (notificações)
- **Vaul**: Componente de Drawer para React
- **embla-carousel-react**: Carrossel
- **input-otp**: Componente de entrada para OTP
- **cmdk**: Componente de menu de comando (rápido e componível)

## Ferramentas de Desenvolvimento
- **ESLint**: Linter para JavaScript/TypeScript
