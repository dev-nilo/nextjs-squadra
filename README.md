# Squadra

Aplicacao para gerenciamento de cartas de jogadores e sorteio de times balanceados.

## Desenvolvimento

1. Instale as dependencias:
   - `npm ci`
2. Execute em modo de desenvolvimento:
   - `npm run dev`
3. Acesse:
   - `http://localhost:3000`

## Build de producao

- `npm run build`
- `npm run start`
